# iGaming KPI Analysis

This project demonstrates a simple data analysis workflow for an online gaming platform.

The goal is to explore key business KPIs and user activity patterns using SQL.

Tools used

SQL
CSV dataset
Basic KPI analysis

Dataset fields

user_id
deposit
bet_amount
win_amount
date

KPIs analysed

Total deposits
Betting volume
Gross Gaming Revenue (GGR)
Top players
Daily betting activity

Project structure

data/
SQL queries/
analysis notes/

This project demonstrates the ability to explore structured datasets, write analytical SQL queries, and extract business insights.
